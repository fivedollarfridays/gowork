# Objective

Create `backend/app/modules/documents/resume_builder.py`. `generate_resume(session_id, target_job_id=None, format="markdown")`. Port `rank_projects` (from `ops:lib/resume_generator.py`) + `extract_keywords` (from `ops:lib/resume_keywords.py`). LLM path gated by `ENABLE_AI_GENERATION`. Deterministic fallback template always available. Prompt-injection defense: blocklist check on worker free-text fields (name, notes, work history) — on match, return deterministic-template path and set `generation_method=template` with logged reason.

# Implementation Plan

1. Port `extract_keywords(job_desc)` + `rank_projects(projects, keywords)`.
2. Implement deterministic template path first (no LLM); returns valid resume.
3. Gate LLM path behind `ENABLE_AI_GENERATION` feature flag; fall back to template on disabled or on injection match.
4. Build `injection_filter` module with blocklist (role-play, "ignore previous", "system:", delimiter injection).
5. Persist `resume_versions` row with counter + `generation_method` column (llm | template).
6. Include cleared-barrier framing where relevant.
7. Write tests: with-job + without-job variants, fallback, injection defense (assert LLM NOT called).
