# Objective

Create `backend/app/modules/documents/cover_letter_builder.py`. `generate_cover_letter(session_id, job_match_ref, resume_version_id)`. Reads employer fair-chance status from `backend/app/modules/criminal/fair_chance_index.py` (existing module — NOT a separate JSON). Addresses record only when employer is fair-chance; else suppresses. Shares the T12.15 prompt-injection defense via the `injection_filter` module.
