1. Install `react-big-calendar` and wire month view.
2. Build list view for upcoming 30 days with AppointmentCard components.
3. Build create/edit/cancel/reschedule modal.
4. Implement PlaceholderPrompt for pathway-generated placeholders.
5. Add all strings to `locales/en.json` + `locales/es.json`; enforce via ESLint (no hardcoded strings in .tsx).
