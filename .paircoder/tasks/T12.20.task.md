# Objective

Create `backend/app/modules/engagement/digest_composer.py`. `compose_digest(session_id, for_date)` builds structured daily email: yesterday (attended appointments, filed applications, cleared barriers), today (upcoming appointments, scheduled actions), this-week (newly listed fair-chance jobs, upcoming recert windows), stall alerts. Ports carryforward + dedupe + notify helpers from ops:lib. Worker first name sourced from `sessions.profile` JSON field.
