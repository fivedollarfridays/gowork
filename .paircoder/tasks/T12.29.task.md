# Objective

Create `frontend/src/app/daily/page.tsx` showing today's digest rendered in-app. Components: `DigestYesterdaySection.tsx`, `DigestTodaySection.tsx`, `DigestWeekSection.tsx`, `StallAlert.tsx`. Home redirect: `/` detects session via `useSession`; if session exists AND `assessment_complete=true`, redirect to `/daily`. Unauthenticated or mid-assessment users see current home.

2. Build four section components (yesterday/today/week/stall-alert) with collapse/expand.
3. Make stall alerts clickable → barrier deep-link.
4. Modify `/` to check `useSession`; redirect to `/daily` if session + `assessment_complete=true`; leave unauth + mid-assessment unaffected.
5. Add all strings to locale files.
