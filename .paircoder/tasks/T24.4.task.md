# Objective

`GET /api/employers/claim/verify?token=…` validates the claim token, marks it used, creates the listing_verifications row with the tier determined by T24.3's domain heuristic, sets a signed `gw_employer_account` cookie binding the browser to the employer identity. 401 uniform across invalid/expired/used; 409 cross-account-conflict.
