# Objective

Extend the S22 `useAccount()` hook with a roles list (requires backend `/api/auth/me` to start returning roles). Add `<RoleGate roles={[...]}>` wrapper component for protected page content. Top nav shows the "Reviewer Dashboard" link only for accounts with at least one reviewer role.
